# 🎲 Snake and Ladder - Console Based Game 

A dynamic, console-based Snake and Ladder game to practice and demonstrate concepts of **Object-Oriented Programming (OOP)** and **Low-Level Design (LLD)** using Java.

---

### 🧾 Features
- 👥 Add multiple players  
- 🎲 Roll dice with automatic extra turn on maximum roll  
- 🐍 Snakes and 🪜 ladders with dynamic board setup  
- 🔄 Undo and redo moves  
- 🏆 Track winner sequence and player rankings  
- 📊 View player positions during gameplay  

---

### 📚 Learning Goals

This project helped me reinforce:  
- SOLID principles & clean OOP architecture  
- Java collections and stack-based move tracking  
- Undo/Redo functionality for game state management  
- LLD practices like modular design, separation of concerns, and scalable gameplay logic  

---

### 🛠️ Tech Stack
- **Language**: Java  
- **Design Concepts**: OOP, Modular Design, LLD principles  
- **Data Handling**: In-memory (no external DB)

### 🧠 Design Patterns Used
- ✅ Stack-based State Tracking: Undo/Redo moves efficiently  
- ✅ Service Pattern: Clean separation of board, game, player, and move logic  
- ✅ OOP Principles: Encapsulation and modular classes for maintainability  

---

### 🧩 System Design Components
- `Game` – Handles player turns, dice rolls, undo/redo, and win sequence  
- `Board` – Represents the board with cells, snakes, and ladders  
- `Cell` – Represents a single cell with optional snake or ladder  
- `Player` – Stores player information and current position  
- `Dice` – Simulates dice rolls  
- `Move` – Tracks player moves for undo/redo operations
